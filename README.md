# MyOperatingSystem
This project runs on a Unix-like Enviornment. If you are using a window, you can use a unix enviorment in an virtual machine
Tools used include:
        ◼️ Make
        ◼️ nasm
        ◼️ C#
        ◼️ qemu-system-x86 (for testing)

Building Instructions
   run Make

   tun ./run.sh
